# CodeXGLUE Code-to-Text

## Dataset

- **Dataset:** CodeXGLUE (code-to-text tasks)
- **Source:** https://github.com/microsoft/CodeXGLUE

## Description

CodeXGLUE code-to-text tasks evaluate code understanding and generation capabilities across multiple programming languages. The model is tasked with generating natural language descriptions from source code.

## Tasks

The benchmark includes code-to-text tasks for 6 programming languages:
- code2text_go: Go
- code2text_java: Java  
- code2text_javascript: JavaScript
- code2text_php: PHP
- code2text_python: Python
- code2text_ruby: Ruby

## Metrics

- Smoothed BLEU-4 score

## Language

Code (multiple programming languages)
