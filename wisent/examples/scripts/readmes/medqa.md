# MedQA

## Dataset

- **Dataset:** GBaker/MedQA-USMLE-4-options-hf
- **Source:** https://huggingface.co/datasets/GBaker/MedQA-USMLE-4-options-hf

## Description

MedQA is a multiple choice question answering dataset based on the United States Medical License Exams (USMLE). It tests specialized medical domain knowledge.

## Task

- **Task:** medqa_4options
- **Type:** Multiple choice (A, B, C, D)

## Metrics

- Accuracy (acc)
- Normalized accuracy (acc_norm)

## Language

English
