# MedMCQA

## Dataset

- **Dataset:** medmcqa
- **Source:** https://huggingface.co/datasets/medmcqa

## Description

MedMCQA is a multiple-choice question answering dataset for medical domain knowledge assessment. It contains medical multiple choice questions assessing detailed medical knowledge.

## Task

- **Task:** medmcqa
- **Type:** Multiple choice (A, B, C, D)

## Metrics

- Accuracy (acc)
- Normalized accuracy (acc_norm)

## Language

English
