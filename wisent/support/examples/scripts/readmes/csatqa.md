# CSATQA

## Dataset

- **Dataset:** HAERAE-HUB/csatqa
- **Source:** https://huggingface.co/datasets/HAERAE-HUB/csatqa

## Description

CSATQA (College Scholastic Ability Test Question Answering) is a Korean benchmark containing SAT-style multiple choice questions for academic assessment.

## Tasks

The benchmark includes 6 subtasks:
- csatqa_gr: Grammar
- csatqa_li: Listening
- csatqa_rch: Reading Comprehension (Hard)
- csatqa_rcs: Reading Comprehension (Standard)
- csatqa_rcss: Reading Comprehension (Super Standard)
- csatqa_wr: Writing

## Metrics

- Accuracy (acc)
- Normalized accuracy (acc_norm)

## Language

Korean
